This cart may require a bootloader file to load YSMenu! You can find it in `/YSMenu/DEMON_common`.

YSMenu instructions (without bootloader) (please create an issue if this does not boot!):
  1. download R4i-SDHC YSMenu
  1. extract R4i-SDHC YSMenu to SD

YSMenu instructions (with bootloader):
  1. download R4i-SDHC YSMenu
  1. extract R4i-SDHC YSMenu to SD
  1. download `R4.dat` from `/YSMenu/DEMON_common` and replace the one extracted to SD
  1. rename `YSMenu.nds` to `_BOOT_DS.nds`
